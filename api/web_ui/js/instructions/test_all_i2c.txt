<h2>

    Bus 1 I2C Addresses:
</h2>
<p>In the 'ALL I2C addresses' section, you will find the addresses for I2C devices connected to Bus 1. Note that these addresses only appear if the respective devices are actively connected:</p>
<ul>
    <li>ADC1: 0x49</li>
    <li>ADC2: 0x4B</li>
    <li>I2C 3v3 Connector: 0x48</li>
    <li>Thermistor Bar Connector: 0x4A</li>
    <li>Band Heater: 0x60 <em>(Note: If the Band Heater appears as 0x61, please report this to Andy, Gabo, or Oscar.)</em></li>
    <li>I2C GPIO Expander: 0x20</li>
    <li>Plunger: 0x48</li>
</ul>
